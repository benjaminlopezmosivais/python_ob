# 3️⃣ Leer archivos grandes (sin gastar memoria)

Si haces esto:

contenido = archivo.read()

Python **carga todo el archivo en memoria**.

Si el archivo mide **2GB o más**, puede ser un problema.

---

## Forma correcta: leer línea por línea

with open("archivo_grande.txt", "r") as archivo:  
    for linea in archivo:  
        print(linea)

Python:

lee una línea  
la procesa  
lee otra línea

Esto usa **muy poca memoria**.

---

## Leer en bloques (chunks)

Otra técnica:

with open("archivo_grande.txt", "r") as archivo:  
    while True:  
        bloque = archivo.read(1024)  
  
        if not bloque:  
            break  
  
        print(bloque)

Aquí se leen **1024 bytes a la vez**.

# 4️⃣ Ejemplo real (procesar logs)

Archivo `logs.txt`:

ERROR conexion  
OK login  
ERROR base_datos  
OK logout

Programa:

errores = 0  
  
with open("logs.txt", "r") as archivo:  
    for linea in archivo:  
        if "ERROR" in linea:  
            errores += 1  
  
print("Errores:", errores)

Salida:

Errores: 2
